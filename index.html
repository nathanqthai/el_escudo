EL Escudo Sequencer

The EL Sequencer is a board that contains an Arduino-compatible microcontroller, and circuitry for controlling up to eight strands of electroluminescent (EL) wire. This code provides a framework for writing code C++ and uploading it to the board without the Arduino IDE.
